CONSIGNES

Nous avions pour mission de créer un rapport web en utilisant uniquement HTML et CSS. Le site devait comporter plusieurs pages : une page d'accueil avec une présentation du groupe et des liens vers les chapitres. Chaque mission (1, 2 et 3) était détaillée dans des sections spécifiques, avec des contenus sur les données de KDou, des analyses et des indicateurs, accompagnés de résultats et de requêtes SQL. Le site devait être responsive, lisible sur ordinateur de bureau et portable. Il fallait respecter les bonnes pratiques HTML5, en utilisant des balises sémantiques et en évitant les éléments superflus. La mise en page devait inclure une navigation fixe, un pied de page, ainsi que des images, des tableaux et des listes. Les styles CSS devaient être bien structurés et le design soigné, avec des tests sur plusieurs navigateurs pour garantir la compatibilité.

HTML, CSS, LE WEB : c'est quoi ?

Le développement web consiste à créer des sites et applications en ligne, combinant des éléments de structure, de style, et logique pour une expérience utilisateur fluide. Le HTML (HyperText Markup Language) est la base d'une page web, fournissant la structure en organisant le contenu avec des balises pour les titres, paragraphes, images, etc. CSS (Cascading Style Sheets) vient ensuite pour styliser cette structure, ajoutant des couleurs, des polices, et des mises en page afin de rendre le site attrayant et professionnel. Ensemble, HTML et CSS permettent de concevoir des sites à la fois fonctionnels et visuellement agréables.

LES BASES DE NOTRE RAPPORT

!DOCTYPE Html
La balise !DOCTYPE HTML est une déclaration située en haut d'un document HTML qui informe le navigateur du type de document qu'il va interpréter. Elle est utilisée pour définir le standard de version HTML. Cette déclaration n'affiche aucun contenu sur la page, mais elle permet au navigateur de rendre correctement le document.
HTML
La balise HTML enveloppe tout le contenu de la page, à l'exception de la déclaration !DOCTYPE html. C’est à l’intérieur de la balise que tous les autres éléments HTML, comme le HEAD et le BODY, sont placés. La balise peut aussi inclure un attribut lang pour spécifier la langue du contenu de la page, ce qui aide les moteurs de recherche et les technologies d’assistance à mieux comprendre et traiter la page.
HEAD
La balise en HTML contient des informations sur le document (métadonnées) qui ne sont généralement pas visibles pour l'utilisateur sur la page web. Elle se situe entre la balise et la balise . C’est dans le HEAD que l’on place des éléments tels que le titre de la page, les liens vers des fichiers CSS, etc.
BODY
La balise BODY en HTML est utilisée pour définir la section d'un document HTML qui contient le contenu visible par l'utilisateur sur une page web. Tout ce qui apparaît dans le navigateur (texte, images, vidéos, liens, formulaires, etc.) doit être placé à l'intérieur de cette balise
MAIN
La balise MAIN en HTML est utilisée pour décrire le contenu principal d'un document. Ce contenu est directement lié au sujet ou à la fonctionnalité principale de la page et est essentiel pour la compréhension du site par l'utilisateur. Elle a été introduite dans HTML5 pour offrir une meilleure structuration et faciliter l’accessibilité.
ARTICLE
La balise ARTICLE en HTML est utilisée pour définir un contenu autonome et indépendant qui pourrait être distribué ou réutilisé à l'extérieur du site. Un article représente un bloc de contenu complet. Cela peut être un article de blog, une publication sur un forum, une nouvelle, etc. L'élément est principalement utilisé pour les sections de contenu qui peuvent être partagées ou redistribuées indépendamment.
CLASS
La balise class n'est pas une balise HTML, mais un attribut utilisé pour appliquer des styles CSS ou pour cibler des éléments avec JavaScript. Cet attribut permet d'identifier et de regrouper des éléments HTML sous un même nom de classe. Il est extrêmement utilisé pour styliser plusieurs éléments HTML de manière cohérente avec une règle CSS.
ID
L'attribut ID est utilisé pour identifier de manière unique un élément dans une page HTML. Contrairement à la classe (class), qui peut être utilisée pour plusieurs éléments, un id doit être unique dans tout le document, ce qui signifie qu'il ne doit être attribué qu'à un seul élément. Il est très utile pour cibler un élément spécifique via du CSS, du JavaScript ou encore pour les liens d'ancrage dans une page.

MÉTHODES D'INSERTION

1. Insérer un tableau en HTML

Le tableau en HTML se construit avec plusieurs balises de base :

  • TABLE : La balise principale qui englobe tout le tableau.
  • TR (table row) : Définit une ligne du tableau.
  • TH (table header) : Définit une cellule d'en-tête (en gras et centré par défaut).
  • TD (table data) : Définit une cellule de données normale.
  • CSS : Espacement interne (padding) et externe (margin) : Le padding est utilisé pour ajouter de l’espace à l’intérieur de chaque cellule.

2. Insérer une image en HTML

Pour insérer une image, on utilise la balise "img". Cette balise est auto-fermante et nécessite au minimum l'attribut src pour indiquer le chemin de l'image et l'attribut alt pour fournir un texte alternatif.

  • (src) : Chemin vers l'image (peut être une URL ou un chemin relatif).

  • (alt) : Texte alternatif qui décrit l'image, utile pour l'accessibilité et si l'image ne peut pas être affichée.
  • (width et height) : Attributs optionnels pour définir la largeur et la hauteur de l'image.

3. Insérer une vidéo en HTML

Pour insérer une vidéo, on utilise la balise "video". Cette balise peut contenir plusieurs attributs et balises pour un meilleur contrôle et accessibilité.

  • (Controls) : Cet attribut ajoute des contrôles de lecture (play, pause, volume, etc.) pour l'utilisateur.
  • (Source) : Cette balise est utilisée pour spécifier les différents formats de la vidéo afin de garantir la compatibilité avec différents navigateurs.
  • (Type) : Spécifie le type de fichier vidéo (par exemple, video/mp4 ou video/ogg).
